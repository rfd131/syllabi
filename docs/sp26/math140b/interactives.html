Interactive Explorations

These browser-based tools let you explore mathematical models hands-on. Adjust parameters, watch eigenvalues move across the complex plane, and build intuition for how matrix structure governs dynamical behavior. No software installation is required.

Population Model Explorer

Explore iterated matrix population models u(t+1) = M u(t). Adjust matrix entries and watch eigenvalues cross the unit circle, flipping dynamics between growth and decay. Includes Leslie population models, stage-based models with stasis, and Markov succession models.

Rotation Matrix Explorer

Explore 2D iterated maps with scaled rotation matrices M = r·R(θ). See how the modulus r controls growth or decay while the angle θ sets the rotation. Watch orbits trace circles, spirals, or quasiperiodic paths in the phase plane.
