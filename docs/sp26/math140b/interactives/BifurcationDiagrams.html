Bifurcation diagrams for inflow–outflow models

Scalar ODEs of the form dy/dt = f(y) − g(y) model a quantity balanced by an inflow and an outflow. Equilibria sit where the two rate curves cross; as a parameter slides, those crossings move, merge, or split — giving the saddle-node, transcritical, and pitchfork bifurcations. The three bifurcations
As a parameter slides, equilibria can collide, exchange stability, or split. The sketches below show qualitative shapes of the branches near each bifurcation point (parameter on the horizontal axis, y* vertical).

Saddle-node (fold)

Two equilibria — one stable, one unstable — collide and annihilate. Below threshold there are two; above, none.

Normal form: dy/dt = r + y² for r < 0 → two roots; r = 0 → tangent; r > 0 → no roots.

y* │ ───● stable │ ╱ │ ● fold point │ ╲ │ ─ ─ ─ unstable └──────────── p

Transcritical

Two branches cross and exchange stability. The same two equilibria persist on both sides; which one is stable flips.

Normal form: dy/dt = r·y − y². Branches y = 0 and y = r cross at the origin.

y* │ ─── y = r stable │ ╱ │ ─ ─ ─╳─ ─ ─ y = 0 unstable (right) │ ╱ │ ─── stable (left) └──────────── p

Pitchfork

One equilibrium loses stability and two new ones are born symmetrically on either side.

Supercritical form: dy/dt = r·y − y³. For r < 0 only y = 0 (stable); for r > 0, y = 0 becomes unstable and y = ±√r appear stable.

y* │ ___ +√r stable │ ╱ │ ───── ─ ─ ─ y = 0 │ ╲ │ ‾‾‾ −√r stable └──────────── p
