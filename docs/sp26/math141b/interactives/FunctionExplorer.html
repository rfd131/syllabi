Function Explorer

Pick a scenario (E. coli populations, atmospheric CO2, a vertical-line-test example, a data table, or a custom f). Drag the x-probe to read f(x); drag the y-target to find all x with f(x) = c or where f(x) > c. Counting how many times a vertical line hits a curve is the vertical line test; it answers whether y is a function of x.
